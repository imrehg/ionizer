Ionizer: Build and run Aluminizer

Microsoft Windows XP / Visual C++

1. Setup build environment
2. From ionizer/aluminizer run qupdate.bat
3. In VC++ open, build, and run ionizer/aluminizer/Aluminizer.sln
4. From ionizer/aluminizerSIM run qupdate.bat
5. In VC++ open, build, and run ionizer/aluminizerSIM/AluminizerSim.sln

Linux (Ubuntu)

1. Setup build environment
2. From ionizer/trlib run: make
3. In Qt Creator open, build, and run ionizer/aluminizer/Aluminizer.pro
4. In Qt Creator open, build, and run ionizer/aluminizerSIM/AluminizerSIM.pro
