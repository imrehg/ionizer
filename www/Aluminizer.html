Ionizer: Aluminizer

Purpose

Aluminizer is software to control ion-trap experiments. Currently it controls the NIST Al+/Mg+ clock, and it is being adapted for the Hg+ clock. It's main function is to generate laser pulses with precise timing and frequency to manipulate the ions. Many of the necessary pulse calibrations are automated, and parameter values are maintained intelligently.

Pulse sequences are generated by an embedded processor on the FPGA that sends data to a custom on-chip Verilog peripheral (IP). This custom IP uses a FIFO buffer to store pulse commands with precise timing information and generate pulse sequences with 10 ns timing resolution.
