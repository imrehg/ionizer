Ionizer

Summary:

Control-software for atomic physics experiments in ion traps.

LaserBrothers software automatically re-locks lasers to reference cavities and doubling-cavities to lasers.

Aluminizer software controls the aluminum ion clock. This is a client/server application with a graphical front-end to control an FPGA-based pulse-sequencer. Pulse-sequences are written as C++ programs to allow inheritance. Timing precision is 10 ns. Many calibrations can be performed automatically by the software. This software is written specifically to control the NIST Al+/Mg+ clock. However, effort was made to keep the code general enough that it can be adapted to other tasks. LabAC software controls a laboratory air-conditioning system. THIS CODE HAS NOT YET BEEN UPLOADED.

BullsIon software shows the coordinates of a laser-beam that's on a webcam. THIS CODE HAS NOT YET BEEN UPLOADED.
